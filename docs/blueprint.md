# **App Name**: KYRAT Festival Hub

## Core Features:

- Homepage Hero Animation: GSAP-powered scroll-triggered video animation to introduce the festival.
- Event Listings: Display a catalog of events with details such as name, category, and description.
- Dynamic Gallery: Fetch and display images from Google Drive using the Google Drive API.
- Interactive Schedule: A dynamic schedule display with filtering options to navigate events, featuring a cool timeline.
- Sponsor Showcase: Display sponsors in a tiered layout with logo and link.
- Team Profiles: Present council and committee members with photos and roles using a GSAP stagger animation.

## Style Guidelines:

- Primary color: Teal (#0E7C86) to represent the sea.
- Background color: Desaturated teal (#E0F2F7) for a light, oceanic feel.
- Accent color: Coral orange (#FF6F61) for highlights and interactive elements.
- Font choice: 'Cabinet Grotesk', a decorative sans-serif that is modern and readable. Note: currently only Google Fonts are supported.
- GSAP scroll-triggered sections on the homepage, creating a dynamic user experience.
- Parallax effects and slide-in animations for various sections to add depth.
- Minimal oceanic accents with dark teal and white for contrast.
